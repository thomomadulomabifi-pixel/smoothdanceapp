# Smooth Dance App 🕺💃

**Learn trending dances with beat-synced steps**

Smooth Dance App is a client-side dance learning app with step-by-step instructions, beat-synchronized animations, and royalty-free music.

## Features
- Free & premium dances (PayPal integration)
- Countdown timer: 3…2…1…Go!
- PDF cheat sheet (~200 steps)
- Beat-synced visual cues
- Responsive design for desktop & mobile

## How to Run
1. Clone or download this repository
2. Open `index.html` in any modern browser
3. Click a dance to start learning
